# **App Name**: Frostfolio

## Core Features:

- Hero Section: Hero Section: Animated name, job title, short introduction, and call-to-action button with smooth entrance animations.
- About Section: About Section: Display information about the user with engaging icons and text animations, emphasizing a clean, glassmorphism aesthetic.
- Projects Section: Projects Section: Feature a horizontally scrollable, animated card layout to showcase projects, incorporating subtle gradients and frosted glass effects.
- Contact Form: Contact Form: Implement a contact form with floating labels and animated send feedback, providing a premium and sleek user experience.
- Theme Toggle: Theme Toggle: A seamless toggle that allows the user to switch between light and dark modes, preserving the glassmorphism design.

## Style Guidelines:

- Primary color: Soft, slightly desaturated blue (#A6D1E6) to evoke tranquility and sophistication, aligning with the frosted glass aesthetic.
- Background color: Very light gray (#F5F7FA) for light mode and a dark gray (#2B2D42) for dark mode, both lightly desaturated, enhancing the glassmorphism effect.
- Accent color: Pale, cool violet (#D3A6E6) to highlight interactive elements and calls to action, complementing the blue with a touch of elegance.
- Body and headline font: 'Inter' (sans-serif) for clean and modern readability across all sections.
- Use a fully responsive layout optimized for both desktop and mobile devices. Incorporate the glassmorphism design with frosted glass effects, blurred translucent panels, soft shadows, and subtle gradients across all sections.
- Apply smooth entrance animations (scroll reveal, fade-in, hover glow, button press animation, glass reflection) to make the portfolio visually engaging and interactive.
- Employ clean and simple icons that align with the glassmorphism aesthetic to enhance the user experience in sections like the 'About Me' section.
